# AI-Powered Job Portal with Resume-Based Job Recommendation System

A full-stack web application that automates job matching using Machine Learning and NLP techniques. The system supports multiple user roles and provides intelligent job recommendations based on resume analysis.

---

## 🚀 Features

### 👤 User Roles
- **Admin** – Platform analytics and user management
- **Business** – Post jobs, view applications, rank candidates, send interview emails
- **Job Seeker** – Upload resume, apply for jobs, get AI-based recommendations
- **Employee** – Apply for jobs in other companies

### 🤖 Machine Learning Features
- Resume parsing and skill extraction using NLP
- Job–resume similarity matching
- Match score calculation (percentage-based)
- ML-ranked applicant list for businesses
- Dynamic skill-based job filtering
- Personalized job recommendations

### 📊 Dashboards
- Job Seeker dashboard with profile & ML status
- Business dashboard with ranked applications
- Admin analytics dashboard (users, jobs, applications)

### 📧 Communication
- Secure email invitations for interviews and job offers
- Resume preview and download

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask, SQLAlchemy
- **Frontend:** HTML, CSS, Bootstrap
- **Database:** MySQL
- **Machine Learning:** NLP, TF-IDF, Cosine Similarity
- **Others:** Flask-Mail, Jinja2
